Creating our Azure Servers

Task

I was tasked to,

1. Setup our file servers on Azure

2. Create our Github accounts and downlaod Github Desktop

3. Verify our Web Server installs

4. Setup a link between GitHub, our servers, and our local machines

Solution

My steps:

I had to register for a Microsoft Azure and Github account. From there we logged into our azure instances on our local machines and created a github repository. After that we linked our github repository to our azure instances. We used the "Azure Status" and "Check php" to check the status of our servers and make sure they were running correctly.
